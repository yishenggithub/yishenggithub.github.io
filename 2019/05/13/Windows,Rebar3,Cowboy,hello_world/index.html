在windows下使用rebar3和cowboy输出hello world

Posted by eason on 2019-05-13

在windows环境下使用rebar3 + cowboy在浏览器输出著名的“hello，world”。想使用一下cowboy，找了一些教程，一些是是linux环境下， 一些是旧版本的rebar。Erlang 的项目本身相关的教程总是不多。在网上找到的一遍教程是是最新版本的cowboy，erlang也是不是最新版本的。最新的cowboy因为一些语法需要用新版本的erlang，不然编译会报错。

1. rebar3

github上交rebar3 git clone 下来，地址是https://github.com/erlang/rebar3，双击bootstrap.bat， 生成rebar3和rebar3.cmd这两个文件。便可以将这两个文件复制到任何需要创建和编译项目的地方。
